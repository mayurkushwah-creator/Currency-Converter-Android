# Currency Converter Android App

A simple Android application that converts entered currency amount to INR.

---

## Features
- User input for amount
- Convert to INR button
- Clean and minimal UI
- Modern Android compatibility

---

## Built With
- Java
- XML
- Android Studio

---

## How It Works
User enters amount → taps convert button → app displays converted INR value.
